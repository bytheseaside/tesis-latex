\subsubsection{Spherical Benchmark Error Statistics}

The discretization error of the spherical benchmark model was evaluated by comparing the
finite element solution $u_h$ with the analytical reference solution $u_{\text{exact}}$.
Error statistics were computed over all sampled points for each conductivity configuration
and for two spatial regions: the full domain and an outer region defined as $r \ge 79\,\mathrm{mm}$.

The outer region threshold was selected to include the material interface at
$r_1 = 80\,\mathrm{mm}$ together with a narrow surrounding band, allowing numerical effects
localized near the conductivity discontinuity to be captured in the statistical analysis.

Table~\ref{tab:sphere-error-distributions} reports distribution statistics of the absolute
discretization error for each configuration and spatial region. In addition to these global
measures, the spatial structure of the error was examined for a representative
Dirichlet--Dirichlet case. Figure~\ref{fig:sphere-error-localization} illustrates both the
absolute error field and its radial distribution, highlighting the localized nature of
high-error regions within an otherwise low-error domain.
